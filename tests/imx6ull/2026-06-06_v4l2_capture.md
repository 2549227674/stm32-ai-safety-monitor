## 1. 硬件接入

- 摄像头：USB 免驱摄像头（UVC 1.00）
- USB ID：0bdc:8088 Y Media Corp.
- 接入位置：i.MX6ULL USB Host（USB Hub 1-1.3）

## 2. 板端识别

### lsusb 摘要

```
Bus 001 Device 008: ID 0bdc:8088 Y Media Corp.
```

### dmesg 识别摘要

```
uvcvideo: Found UVC 1.00 device USB Camera (0bdc:8088)
usbcore: registered new interface driver uvcvideo
USB Video Class driver (1.1.1)
```

注意：首次插入时出现多次 disconnect/reconnect（device number 4→5→6→7→8），最终稳定在 device 8。

### /dev/video* 节点

```
crw-rw---- 1 root video 81, 0 Jan  1 00:00 /dev/video0   ← pxp (i.MX6ULL Pixel Pipeline)
crw-rw---- 1 root video 81, 1 Jan  1 00:03 /dev/video1   ← USB Camera (UVC)
```

注意：`/dev/video0` 是 i.MX6ULL 的 pxp_v4l2 设备，不是摄像头。摄像头为 `/dev/video1`。

## 4. 摄像头格式

### 支持的 pixelformat

- MJPG (Motion-JPEG, compressed)

### 支持的分辨率（MJPG）

| 分辨率 | 帧率 |
|---|---|
| 1280x720 | 30 fps |
| 848x480 | 30 fps |
| 800x600 | 30 fps |
| 640x480 | 30 fps |
| 640x360 | 30 fps |
| 352x288 | 30 fps |
| 320x240 | 30 fps |
| 160x120 | 30 fps |

### 选用的格式

- pixelformat: MJPG
- 分辨率: 640x480

## 5. 抓图命令

```bash
mkdir -p /opt/edge-ai-safety-monitor/captures
v4l2-ctl -d /dev/video1 \
  --set-fmt-video=width=640,height=480,pixelformat=MJPG \
  --stream-mmap=3 \
  --stream-count=1 \
  --stream-to=/opt/edge-ai-safety-monitor/captures/task04_latest.jpg
```
