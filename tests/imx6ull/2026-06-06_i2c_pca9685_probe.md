## 4. 板端 I2C 工具检查

命令：

```bash
ssh root@192.168.137.110 "ls -l /dev/i2c-* 2>/dev/null || true; which i2cdetect || true"
```

输出：

```text
crw------- 1 root root 89, 0 Jan  1 00:00 /dev/i2c-0
crw------- 1 root root 89, 1 Jan  1 00:00 /dev/i2c-1
/usr/sbin/i2cdetect
```

结论：

| 项目 | 结果 |
|---|---|
| `/dev/i2c-*` | `/dev/i2c-0`、`/dev/i2c-1` |
| `i2cdetect` | 存在：`/usr/sbin/i2cdetect` |
| 扫描方式 | 直接使用 `i2cdetect -y <bus>` |
| `i2c_scan` | 未构建、未部署；板端已有 `i2cdetect`，不需要 fallback |
