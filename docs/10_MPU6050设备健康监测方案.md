# 10 MPU6050 设备健康监测方案

## 10.1 模块定位

GY-521 MPU6050 是个人已拥有材料，可优先用于前期打样。它在项目中用于检测设备柜被敲击、搬动、异常震动或风扇震动增强。震动异常也会被纳入事件上报和云端 AI 分析的多模态数据中。

---

## 10.2 接线建议

| GY-521 引脚 | STM32 引脚 | 说明 |
|---|---|---|
| VCC | 3.3V 或 5V | 视模块带不带稳压 |
| GND | GND | 必须共地 |
| SCL | PB6 | I2C1_SCL，与 OLED 共线 |
| SDA | PB7 | I2C1_SDA，与 OLED 共线 |
| AD0 | GND | 地址通常 0x68 |

---

## 10.3 为什么不优先做 FFT

STM32F103C8T6 资源有限；两周课程项目更需要稳定演示；RMS、峰峰值、方差更容易答辩讲清楚；敲击、搬动、风扇震动用轻量算法即可识别。

---

## 10.4 采样频率和窗口

| 参数 | 建议值 |
|---|---|
| 采样频率 | 50 Hz |
| 窗口长度 | 50 点（1 秒） |
| 基线校准 | 上电静止 2 秒，采集 100 点 |
| 报警确认 | 连续 2-5 个窗口超阈值 |

---

## 10.5 算法指标

| 指标 | 公式 | 用途 |
|---|---|---|
| 均值 mean | Σx/N | 判断姿态或持续偏移 |
| 峰峰值 pp | max(x)-min(x) | 判断敲击、突发震动 |
| RMS | sqrt(Σx²/N) | 判断持续震动强度 |
| 方差 var | Σ(x-mean)²/N | 判断波动程度 |
| 持续计数 | 连续超阈值窗口数 | 避免偶然碰撞误报 |

---

## 10.6 C 语言伪代码

```c
void MPU_ProcessSample(float ax, float ay, float az)
{
    float dx = ax - base_ax;
    float dy = ay - base_ay;
    float dz = az - base_az;
    float mag = sqrtf(dx*dx + dy*dy + dz*dz);

    mpu_buf.mag[mpu_buf.index++] = mag;
    if (mpu_buf.index >= MPU_BUF_SIZE) {
        mpu_buf.index = 0; mpu_buf.full = 1;
    }
    if (!mpu_buf.full) return;

    metrics.mean = CalcMean(mpu_buf.mag, MPU_BUF_SIZE);
    metrics.rms  = CalcRMS(mpu_buf.mag, MPU_BUF_SIZE);
    metrics.pp   = CalcPeakToPeak(mpu_buf.mag, MPU_BUF_SIZE);
    metrics.var  = CalcVariance(mpu_buf.mag, MPU_BUF_SIZE, metrics.mean);

    if (metrics.rms > VIB_RMS_TH && metrics.pp > VIB_PP_TH)
        vib_over_cnt++;
    else if (vib_over_cnt > 0)
        vib_over_cnt--;

    metrics.alarm = (vib_over_cnt >= VIB_CONFIRM_CNT);
}
```

## 10.7 异常触发条件

| 异常 | 条件 | 风险处理 |
|---|---|---|
| 敲击 | pp 突然超阈值，连续 2 个窗口 | +2 或 +3 |
| 搬动 | 均值持续偏移超阈值 1 秒以上 | +3 |
| 风扇异常震动 | RMS 持续高于基线 5 秒 | +2 |
| 偶然碰撞 | 仅一个窗口超阈值 | 只记录，不报警 |

---

## 10.8 与 AI 层的结合

MPU6050 的震动数据（vib_rms_g、vib_pp_g）包含在事件上报的 sensor_snapshot 中，云端 AI 可以结合图像和震动数据给出更准确的异常解释。例如当图像中看到物体位移，且 MPU6050 检测到搬动，AI 可以判断为"设备被移动"。

---

## 10.9 答辩演示方案

| 演示 | 操作 | 现象 |
|---|---|---|
| 敲击检测 | 轻敲柜体侧板 | RMS/PP 增大，进入预警或报警 |
| 搬动检测 | 抬起或移动模型 | 姿态偏移，显示 Move Detected |
| 风扇震动 | 将小风扇靠近或固定在板上 | RMS 持续升高 |
| 误报抑制 | 单次轻碰 | OLED 指标变化但不进入报警 |

---

## 10.10 误报规避

上电静止校准。连续窗口超阈值才报警。单次尖峰只记录不改变主状态。MPU6050 风险分不直接进入 DANGER，除非与门磁、烟雾、火焰等组合。I2C 读取失败时标记 MPU_FAULT，不阻塞主线。
