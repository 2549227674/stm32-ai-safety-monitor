# 12 材料清单与获取计划

## 12.1 材料来源原则

| 分类 | 含义 |
|---|---|
| 个人已拥有材料 | 已在手，可优先用于前期打样、功能验证、结构验证 |
| 待获取材料 | 当前个人没有但项目需要，优先从实验室领取或借用；若没有则小组自行补购 |
| 最终上交实物所需材料 | 建议最终作品使用可独立留存的一套器件，避免上交个人材料 |
| 可替代材料 | 当目标器件缺失或不稳定时，可替换完成同等演示目的 |
| 不建议使用材料 | 存在安全、跑题、调试成本高或不符合主线的问题 |

## 12.4 最终上交实物建议

| 器件 | 必须性 | 说明 |
|---|---|---|
| STM32F103C8T6 核心板 | 必须 | 唯一主控 |
| OLED、蜂鸣器、RGB、按键 | 必须 | 人机交互 |
| MQ-2、火焰、PIR/门磁 | 推荐必须 | 传感器输入 |
| 风扇或继电器 + 驱动模块 | 必须至少一个 | 执行器动作 |
| ESP32-CAM | 推荐 | 抓拍增强 |
| 5V 3A 电源 | 强烈推荐 | 稳定供电 |
| 亚克力板/洞洞板/固定材料 | 必须 | 实物固定 |

## 12.6 不建议使用材料

| 材料/功能 | 原因 |
|---|---|
| 激光模块 | 安全风险和主题不匹配 |
| USB 免驱摄像头作为最终核心 | 依赖 PC，不是嵌入式核心 |
| 12V 锂电池作为主供电 | 安全和稳定性风险 |
| 真实高压强电负载 | 课堂安全风险高 |
| PCB 打样 | 周期风险 |

## 12.8 获取顺序建议

1. 先确认基础安全闭环层所有材料（第 1-3 天）。
2. 利用个人已有材料先打样（按键、KY-016、温湿度、GY-521）。
3. 本地闭环稳定后获取 ESP32-CAM（第 4-5 天）。
4. 服务器和 AI 不需要额外硬件采购，只需笔记本和网络。
5. microSD、MQTT、水泵真实喷淋最后考虑。
