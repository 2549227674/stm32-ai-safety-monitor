# i.MX6ULL Stage - Docs Archive

This directory originally contained i.MX6ULL-era design documents.

These documents have been archived as historical evidence. The current mainline docs are in `docs/` (root level) and have been rewritten for the OPi5 single-board architecture.

## Key Historical Documents

- `docs/03_硬件系统设计与供电安全_iMX6ULL_OPi5.md` — original dual-board hardware design
- `docs/08_iMX6ULL视觉采集_云台_Web服务器方案.md` — original V4L2 + pan/tilt design

These files are preserved in git history. See `git log --all -- docs/03_硬件系统设计与供电安全_iMX6ULL_OPi5.md` for the original content.
