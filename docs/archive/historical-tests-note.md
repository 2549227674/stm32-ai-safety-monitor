# Historical Tests Note

Test evidence from the i.MX6ULL stage is preserved in:

- `tests/imx6ull/` — i.MX6ULL hardware verification (GPIO, I2C, PWM, sensors, actuators)
- `tests/integration/` — end-to-edge integration tests (imx_safetyd, vertical slice)
- `tests/opi5/` — OPi5 AI service tests (mock, Qwen3-VL, WiFi, systemd)
- `tests/legacy/` — STM32/ESP32-CAM era tests

These records serve as engineering iteration evidence for the thesis defense. They are not part of the current OPi5 demo mainline.
