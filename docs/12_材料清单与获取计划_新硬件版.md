# 12 材料清单与获取计划（新硬件版）

## 12.1 材料来源原则

优先使用现有器件和低压演示负载，不新增复杂高风险材料。所有材料需服务 MVP：i.MX6ULL 控制、OPi5 推理、V4L2 抓拍、PCA9685 云台、Flask 展示。

## 12.2 已有核心板卡

| 材料 | 用途 | 状态 | 备注 |
|---|---|---|---|
| i.MX6ULL Pro | Linux 本地安全控制与视觉采集 | 已计划使用，待验证 | SDK/SSH/USB Host 需确认 |
| Orange Pi 5 16GB + 128G NVMe | 本地 RKNN 推理与存储 | 已计划使用，待验证 | 模型与依赖需调查 |
| PC / WSL | 开发、交叉编译、部署 | 已计划使用，待配置 | SDK 路径待填 |

## 12.3 新增器件

| 器件 | 用途 | 是否 MVP 必需 | 注意事项 |
|---|---|---|---|
| 16 路 PWM 驱动板（PCA9685 类） | MG90 舵机 PWM | 推荐 | I2C 电平和地址需验证 |
| MOS 管/模块 | 风扇、水泵、继电器等低压负载 | 推荐 | 栅极下拉、负载独立供电 |
| MG90 二维云台 | 摄像头巡检 | 推荐 | 舵机独立 5V~6V 供电 |
| 1080p 免驱 USB 摄像头 | 图像采集 | 必须 | V4L2 格式需验证 |
| 门磁/PIR/火焰/MQ2 | 本地传感器 | 至少一路必须 | 根据已有器件选择 |
| MPU6050/GY-521 | 设备健康监测 | 加分项 | 非 MVP |

## 12.6 线材与焊接材料

- 杜邦线：短距离临时调试。
- 端子线：负载和供电。
- 细导线/焊锡：必要时固定引出测试点。
- 热缩管/绝缘胶带：防短路。
- 扎带/胶柱：固定板卡与线束。

## 12.7 替代方案

| 原计划 | 替代 |
|---|---|
| 真实火焰/烟雾 | 使用按键或模拟输入触发 ALARM，避免危险 |
| 真实水泵 | LED/小风扇替代低压负载 |
| 真实 RKNN 模型 | mock AI 服务 + 已有 demo |
| 云台不稳 | 固定摄像头 + 单角度抓拍 |

## 12.8 不建议使用材料

- 220V 继电器负载。
- 未知规格大电流电机。
- 无下拉/无隔离的裸 MOS 接线。
- 直接从板卡 IO 或 5V 给舵机供电。
- 长线无共地的信号连接。

## 12.9 上交实物建议

上交/展示时应带：i.MX6ULL、OPi5、USB 摄像头、PCA9685、MG90 云台、低压负载、降压模块、逻辑分析仪波形截图、Dashboard 截图。
