# 12 材料清单与获取计划（OPi5 版）

## 12.1 材料来源原则

优先使用现有器件和低压演示负载，不新增复杂高风险材料。所有材料服务 OPi5 一板主控 + AI 推理 + Flask 展示。

## 12.2 核心板卡

| 材料 | 用途 | 状态 |
|---|---|---|
| Orange Pi 5 16GB + 128G NVMe | 主控 + AI 推理 | 当前主线 |
| PC / WSL | 开发、Flask 后端、React 前端 | 已有 |

## 12.3 传感器

| 器件 | 用途 | 引脚 | 状态 |
|---|---|---|---|
| PIR 人体红外模块 | 运动检测 | pin13 / gpio139 | 当前主线 |
| 火焰传感器模块 | 火焰检测 | pin15 / gpio28 | 当前主线 |
| MQ-2 烟雾传感器 | 烟雾/气体检测 | pin19 / gpio49 | 当前主线 |
| DHT11 温湿度 | 未来外置 MCU 预留 | — | 仅 mock |

## 12.4 输出设备

| 器件 | 用途 | 引脚 | 说明 |
|---|---|---|---|
| RGB LED + 220Ω 限流电阻 | 状态指示灯 | pin21/23/24 | 共阴，active HIGH |
| 有源蜂鸣器 + NPN 三极管 | 声音报警 | pin26 / gpio35 | active LOW |
| Water MOS 模块 | 低压喷淋开关 | pin11 / gpio138 | active HIGH |
| MG90 单轴 Pan 舵机 | 摄像头水平扫描 | pin7 / PWM15 | 独立 5V~6V 供电 |

|---|---|---|

## 12.7 供电模块

| 模块 | 用途 | 说明 |
|---|---|---|
| 12V 适配器/电池 | 总电源 | — |

## 12.8 调试工具

| 工具 | 用途 |
|---|---|
| 逻辑分析仪 | GPIO/I2C/PWM 波形验证 |
| 万用表 | 电压/电流/共地确认 |
| 杜邦线/端子线 | 临时调试接线 |

## 12.9 线材与辅材

- 杜邦线：短距离临时调试
- 端子线：负载和供电
- 细导线/焊锡：必要时固定引出
- 热缩管/绝缘胶带：防短路
- 扎带/胶柱：固定板卡与线束

## 12.10 不再使用的材料

以下材料属于历史阶段，不在当前 OPi5 主线 BOM 中：

| 材料 | 历史阶段 | 说明 |
|---|---|---|
| i.MX6ULL Pro | 第二版双板架构 | 已迁移到 OPi5 |
| STM32 + ESP32-CAM | 第一版端云架构 | 已归档 |
| PCA9685 16路 PWM | 第二版 i.MX 控制 | 模块问题，已弃用 |
| MG90 Tilt 舵机 | 第二版二维云台 | 当前仅单轴 Pan |
| 继电器模块 | 第二版 P1 扩展 | 不在当前 BOM |
| 风扇 | 第二版 P1 扩展 | 不在当前 BOM |
| P1 水泵 | 第二版 P1 扩展 | 当前用 Water MOS 替代 |

## 12.11 不建议使用

- 220V 继电器负载
- 未知规格大电流电机
- 无下拉/无隔离的裸 MOS 接线
- 直接从板卡 IO 或 5V 给舵机供电
- 长线无共地的信号连接
