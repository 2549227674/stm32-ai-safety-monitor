# 02 需求分析与功能优先级

## 2.1 功能需求按四层组织

### 第一层：基础安全闭环层

| 编号 | 功能 | 说明 |
|---|---|---|
| F01 | STM32 主控多传感器采集 | MQ-2 烟雾、火焰、PIR、门磁、温湿度，全部由 STM32 采集 |
| F02 | 多模态风险评分引擎 | 六类传感器数据融合加权，生成风险分数和风险等级 |
| F03 | 多级状态机 | INIT → IDLE → ARM_DELAY → NORMAL → PRE_ALARM → ALARM → DANGER → SILENCE → RECOVER → FAULT |
| F04 | OLED 实时显示 | 显示系统状态、风险等级、传感器值、故障标志 |
| F05 | 4 位按键交互 | 布防、撤防、静音、复位，预留长按扩展 |
| F06 | RGB LED 状态灯 | 正常绿、预警黄、报警红、危险红紫、故障蓝闪 |
| F07 | 蜂鸣器分级报警 | 慢响、快响、连续响、静音 |
| F08 | 执行器联动 | 风扇通风、继电器断电保护、水泵喷淋模拟，均由 STM32 控制 |
| F09 | 本地事件日志 | 串口日志输出、事件计数、可选 RAM 环形日志 |

### 第二层：边缘视觉取证层

| 编号 | 功能 | 说明 |
|---|---|---|
| F10 | ESP32-CAM 异常抓拍 | STM32 异常触发 → UART 发送 SNAP_REQ → ESP 抓拍 → 返回结果 |
| F11 | UART ACK/心跳/重传 | SNAP_REQ ACK、PING/PONG 心跳、超时重传、错误码 |
| F12 | 摄像头离线降级 | 连续心跳失败后标记 CAM_OFFLINE，本地报警继续 |
| F13 | 可选 microSD 存图 | 异常图片保存到 SD 卡，SD 失败不影响主线 |
| F14 | MPU6050 设备健康监测 | RMS/峰峰值/方差判断震动异常，参与风险评分 |

### 第三层：端云协同与远程可视化层

| 编号 | 功能 | 说明 |
|---|---|---|
| F15 | Web Dashboard | 当前状态、传感器数值、风险等级、最近事件、抓拍图片 |
| F16 | 手机端浏览 | 手机浏览器/PWA 访问 Web 页面，查看状态和历史 |
| F17 | HTTP 事件上传 | 事件 JSON + 传感器快照上传至服务器 |
| F18 | 图片上传 | 抓拍图片上传至服务器存储 |
| F19 | 服务器事件存储 | Flask/FastAPI 接收事件和图片，SQLite/JSON 存储 |
| F20 | 历史事件查询 | Web/手机端查看历史事件列表和详情 |
| F21 | 可选 MQTT 上报 | MQTT Topic 状态上报和事件上报 |

### 第四层：云端 AI 智能解释层

| 编号 | 功能 | 说明 |
|---|---|---|
| F22 | AI 图像复核 | 将抓拍图片发送给云端 AI 进行视觉分析 |
| F23 | 异常类别识别 | AI 返回异常类型（烟雾/火焰/入侵/正常等） |
| F24 | 场景描述生成 | AI 生成中文场景描述和处理建议 |
| F25 | AI 结果展示 | AI 解释结果在 Web/手机端展示 |
| F26 | AI 失败降级 | AI 超时/失败时 Web 显示"AI 分析不可用"，不影响本地 |

## 2.2 非功能性需求

| 类型 | 要求 |
|---|---|
| 实时性 | 本地报警响应 <1 秒；OLED 刷新 200-500 ms；传感器扫描 50 ms-2 s 分层 |
| 可靠性 | ESP32-CAM、Wi-Fi、Web、AI 失败不影响本地闭环 |
| 可演示性 | 3-5 分钟内展示正常 → 异常 → 报警 → 抓拍 → Web 查看 → AI 解释 → 降级 |
| 可维护性 | 代码分层清楚，驱动、应用、协议、UI 分开 |
| 可调试性 | OLED 状态、RGB 状态、串口日志、错误码可观测 |
| 安全性 | 低压演示，不接 220V；水路远离主控与电源 |
| 可提交性 | 实物固定可靠，源程序、原理图、报告、答辩材料完整 |

---

## 2.3 安全性需求

| 安全点 | 要求 |
|---|---|
| 高压隔离 | 不进行真实高压强电演示，继电器只控制低压模拟负载 |
| 水电隔离 | 水泵、水管、喷头、托盘远离 STM32、ESP32-CAM 和电源模块 |
| 电机驱动 | 水泵/风扇不能由 STM32 GPIO 直接驱动，必须用 MOS 管/驱动板/继电器模块 |
| 反电动势 | 继电器线圈、电机、水泵需要续流保护或使用带保护模块 |
| 电平保护 | MQ-2 AO 接 STM32 ADC 前必须确认不超过 3.3V，必要时分压 |
| ESP 供电 | ESP32-CAM 使用稳定 5V 供电，预留峰值电流 |
| 火焰演示 | 可短时演示，必须准备安全替代触发方式 |

## 2.4 可靠性与降级需求

| 故障 | 降级策略 | 对本地闭环影响 |
|---|---|---|
| ESP32-CAM 离线 | STM32 标记 CAM_OFFLINE，OLED 提示 | 无影响，本地报警继续 |
| Wi-Fi 离线 | ESP 返回 NET_FAIL，OLED 显示网络离线 | 无影响，本地动作不变 |
| 拍照失败 | 返回 SNAP_FAIL，记录失败码 | 无影响，本地报警继续 |
| microSD 失败 | 返回 SD_FAIL，改用 Web latest | 无影响 |
| 服务器不可达 | 上传失败，事件缓存或丢弃 | 无影响 |
| 云端 AI 超时 | Web 显示"AI 分析不可用" | 无影响，不改变风险等级 |
| AI 返回异常 | 忽略 AI 结果，显示 AI_ERROR | 无影响 |
| DHT 读取失败 | 保留上次值或显示传感器故障 | 不阻塞烟雾/火焰报警 |
| MPU6050 失败 | 禁用震动检测 | 其他传感器继续工作 |
| 水泵不可用 | 用风扇、继电器或 LED 模拟 | 无影响 |

## 2.5 四层功能优先级

### 基础安全闭环层（目标：必须完成）

完全不依赖 ESP32-CAM、网络、Web、AI。

| 功能 | 验收标准 |
|---|---|
| STM32 主控采集 | 至少接入 MQ-2、火焰、PIR/门磁、温湿度中的 3 类输入 |
| OLED 显示 | 显示系统状态、风险等级、至少 2 个传感器值 |
| 按键 | 4 位按键实现布防、撤防、静音、复位 |
| RGB LED | 颜色/闪烁与状态对应 |
| 蜂鸣器 | 不同报警等级有不同鸣叫模式，静音键有效 |
| 执行器动作 | 风扇/继电器/水泵至少一个由 STM32 控制并随状态动作 |
| 风险评分 | 多传感器触发时风险等级升高 |
| 状态机 | 能进入 NORMAL、PRE_ALARM、ALARM、DANGER、SILENCE、RECOVER |
| 本地闭环 | 不接 ESP32-CAM 也能完整演示 |

### 边缘视觉取证层（目标：推荐完成）

| 功能 | 验收标准 |
|---|---|
| ESP32-CAM 抓拍 | STM32 触发 SNAP_REQ 后 ESP 返回 ACK 和 SNAP_OK/SNAP_FAIL |
| PING/PONG | STM32 能检测 ESP 在线 |
| 离线检测 | 连续心跳失败后 OLED 显示 CAM_OFFLINE |
| 超时重传 | SNAP_REQ ACK 超时后最多重传 3 次 |
| MPU6050 基础震动 | 敲击/搬动触发震动风险加分 |

### 端云协同与远程可视化层（目标：争取完成）

| 功能 | 验收标准 |
|---|---|
| Web Dashboard | 手机/电脑浏览器能查看状态、传感器值、抓拍图片 |
| 事件上传 | 至少一条事件成功上传至服务器 |
| 图片上传 | 至少一张抓拍图片成功存储在服务器 |
| 手机端查看 | 手机浏览器访问 Web 页面正常 |
| 历史查询 | 能查看最近 5 条历史事件 |

### 云端 AI 智能解释层（目标：争取完成）

| 功能 | 验收标准 |
|---|---|
| AI 请求 | 将事件 JSON + 图片发送给云端 AI |
| AI 返回 | 收到异常类别、场景描述、处理建议 |
| AI 展示 | AI 结果在 Web/手机端页面可查看 |
| AI 降级 | AI 失败时 Web 显示"AI 分析不可用"，本地不受影响 |

## 2.6 暂不建议做的功能

| 功能 | 原因 |
|---|---|
| FOTA | 调试成本高，对课程主线帮助小 |
| 复杂公网云平台强依赖 | 现场网络不可控，会削弱本地闭环 |
| 从零训练 AI 模型 | 时间不可控，不符合两周周期 |
| 在 ESP32-CAM 上跑复杂图像识别 | 算力有限，网络依赖大 |
| FFT 震动分析 | STM32F103C8T6 资源有限，RMS/峰峰值足够展示 |
| 复杂机械臂/小车 | 跑题且结构风险高 |
| PCB 打样 | 周期风险大 |
| 真实高压强电 | 安全风险高 |
| 激光模块正式功能 | 安全和跑题问题 |
| USB 免驱摄像头作为最终核心 | 只能用于 PC 端辅助测试 |
| 原生手机 APP | 手机浏览器 Web 页面/PWA 已够用 |
